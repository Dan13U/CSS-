To CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación de documentos HTML. Es decir, CSS se encarga de dar estilo a las páginas web, definiendo cómo se verán los elementos (colores, tamaños, fuentes, etc.) y cómo se organizarán en la pantalla.

2. Funcionamiento Básico de CSS

CSS funciona añadiendo reglas de estilo a los elementos HTML. Estas reglas se escriben en un archivo separado o directamente dentro del código HTML. El navegador interpreta estas reglas y aplica los estilos correspondientes a los elementos seleccionados.

3. Incluir CSS en un archivo HTML

Hay tres formas principales de incluir CSS en un archivo HTML:

  • En un archivo externo: Se crea un archivo con extensión .css y se enlaza desde el elemento link dentro del head.
  • Dentro del elemento head: Se crea un elemento style y se escriben las reglas de estilo dentro de él.

Ejemplo de enlace a un archivo CSS externo: <link rel="stylesheet" href="styles.css">

4. Definición de una regla CSS

Una regla CSS es una instrucción que define cómo se mostrará un elemento HTML. Se compone de dos partes principales: un selector y un bloque de declaración.

5. Elementos de una regla CSS

Selector: Indica a qué elementos HTML se aplicará la regla. Puede ser un elemento HTML (p.ej., p), una clase (p.ej., .destacado) o un ID (p.ej., #miParrafo).

Propiedad: Es una característica que se puede modificar en un elemento (p.ej., color, font-size).

Valor: Es el valor asignado a la propiedad (p.ej., red, 16px).

6. Medios CSS

Los medios CSS permiten aplicar diferentes estilos a un documento en función del medio en el que se visualiza (pantalla, impresión, etc.). Se utilizan las @media queries para definir reglas específicas para cada medio.

Ejemplo:

@media (max-width: 768px) {
    body {
        font-size: 14px;
    }
}

7. Comentarios CSS

Los comentarios se utilizan para añadir notas explicativas al código CSS sin que afecten al resultado final. Se escriben entre /* y */.

Ejemplo:
/* Este es un comentario de una línea */
p {
color: blue; /* Color del texto */
}

8. Sintaxis de la definición de las propiedades CSS

La sintaxis general de una propiedad CSS es:
propiedad: valor;
color: red;
font-size: 16px;
margin: 10px;

Importante: Los valores de las propiedades pueden ser de diferentes tipos:

  • Palabras clave: none, inherit, auto, etc.
  • Números: Con o sin unidades (px, em, etc.).
  • Colores: Nombres de colores, códigos hexadecimales, RGB, HSL.
